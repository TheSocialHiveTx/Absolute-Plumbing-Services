
# Absolute Plumbing Services - WaaS Output

### A) Project Summary
A high-converting, mobile-first, phone-call-focused lead generation website for Absolute Plumbing Services. The design prioritizes local trust signals, emergency urgency, and clear calls to action (CTA).

### B) Site Structure & Section Order
1.  **Sticky Header**: Logo + Call Now Button
2.  **Hero Section**: Urgent H1, Trust Badge, Primary CTA
3.  **Trust Signals**: Review stars, specific Google review snippets
4.  **Services Overview**: Grid of the top 6 plumbing services
5.  **Why Choose Us**: Benefit-driven icons (Pricing, Experience, Speed)
6.  **Primary Offer / Secondary CTA**: Emergency repair focus
7.  **Service Area**: Map-based text and local relevancy
8.  **FAQ Section**: Handling common objections
9.  **Footer**: Contact info, Business hours, GMB link
